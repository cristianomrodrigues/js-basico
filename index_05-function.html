Função comum Vs. Função anonima

/função comum/ function aoQuadrado(numero) {
return numero * numero;
}
/função anonima/ function (numero) {
return numero * numero;
}
/Arrow function - syntactical sugar / const aoQuadrado = (numero) => numero * numero;

Função comum Vs. Função anonima

/comum/ function aoQuadrado(numero) {
return numero * numero;
}
/anonima/ function (numero) {
return numero * numero;
}
/arrow - quando função retorna apenas um valor / const aoQuadrado = (numero) => numero * numero;
